# HackathonFinderIOS
IOS app that helps users choose Hackathons that meet their specific needs.

* Swift 3
* xCode 8.3

Dependancies:
  * Alamofire Cocoapod - https://github.com/Alamofire/Alamofire
  * Toast-Swift Cocoapod - https://github.com/scalessec/Toast-Swift
